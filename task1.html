✅ Flexbox (Flexible Box Layout) — is a layout system introduced with CSS3. It means "Flexible box layout" and mainly makes it easier to arrange, position, and manage the spacing of elements on web pages. Flexbox can work in both horizontal and vertical directions and ensures that elements are flexibly arranged according to the screen size. Flexbox allows web designers and developers to create more flexible and responsive designs. Using Flexbox, it is possible to easily center elements, distribute spaces evenly, and adapt to different screen sizes.

✅ Why is Flexbox superior?

    Flexbox has several advantages over float and inline-block methods:
    It is easier to center and manage spaces.
    Elements can be distributed evenly or flexibly.
    Order changes can be made only with CSS, not in HTML.
    It is ideal for mobile and responsive design.
